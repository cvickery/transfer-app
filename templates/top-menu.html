CUNY Transfer Application

The following functions are currently available:

Review Existing Rules
There are over a million rules for how courses transfer from one CUNY college to another. The Review Rules function lets anyone look at and comment on existing rules after first narrowing down the set to a manageable size based on sending / receiving colleges and sending / receiving disciplines.
Look Up Rules
Look up all the existing rules that involve a course or set of courses. You can search for rules where the courses are in the sending, receiving, or both parts of rules. You can specify sets of catalog numbers, such as all 200-level courses, by using regular expressions. There is a link on the page to a mini-tutorial on regular expressions.
Regular Expressions
This is the regular expression guide that is referenced from the Look Up Rules page.
Pending Reviews
This is a maintenance utility that shows reviews that have been submitted but not yet confirmed by the submitter. These reviews are purged if not confirmed within two days of submission.
Course Catalogs
This function will display all the courses currently offered at any CUNY College in standard college-bulletin format. But you have to say "Please."
